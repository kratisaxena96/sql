Databases ---> It is used to store large data either in structured form or in non structured form

            Why to use database in comparison with files????

                1. Large Data 
                2. Redundancy (repitition)
                3. Multiple User access database
                4. Connectivity 
                5. Security (Privileges)
                6. ACID
                    A --> Atomicity -->  Either full or nothing
                    C --> Consistency  ---> COnsistent before transaction or after trasaction
                    I --> Isolation --> Isolated from others transaction 
                    D --> Durability
                7. Memory save
                8. Searching or any operation --> Queries
